# Local AI Agent

> An extensible, self-steering local LLM assistant that can optionally enrich answers with iterative, relevance‑filtered DDGS (DuckDuckGo/Bing/Brave) web searches while tracking multi‑topic conversational context.

## Documentation

Full project documentation is available in the `docs/` folder (start at `docs/index.md`).

## Requirements

- Python: **3.12** is the tested and supported interpreter for development and CI (the installer attempts to locate a Python 3.12 interpreter). See `docs/install.md` for details on the installer and Python discovery.
